** Define Font
- Need to use web-font (.woff2) format.

  - nix :: ~nix shell nixpkgs#woff2~ (flakes)
- Used fonts:
  - victor - Victor mono nerd font
  - times - tinos nerd font
  - impress - Impress bt
  - code - are fork of recursive font.
